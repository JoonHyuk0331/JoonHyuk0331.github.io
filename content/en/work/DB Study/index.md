학교에서 데이터베이스를 공부하며, 데이터의 효율적인 저장, 관리, 검색 방법을 배웠습니다. 관계형 데이터베이스 모델을 중심으로 테이블, 열, 행, 키와 같은 기본 개념을 익히고, 이를 활용해 데이터 간의 관계를 정의하고 관리하는 방법을 학습했습니다. SQL 언어를 사용해 데이터베이스를 생성, 수정, 삭제하는 방법과 데이터를 조회하고 조작하는 다양한 쿼리 작성법을 연습했습니다. 또한, 정규화 과정을 통해 데이터의 중복을 줄이고 무결성을 유지하는 방법을 배웠으며, 인덱스를 활용한 성능 최적화 기법도 익혔습니다. 트랜잭션 관리와 ACID 특성을 통해 데이터의 일관성을 유지하는 방법도 학습했습니다. 실습을 통해 ER 다이어그램을 작성하고, 이를 기반으로 데이터베이스를 설계하고 구현하는 경험을 쌓았습니다. 추가로 학습한 내용을 바탕으로 얼마전 SQLD 자격증을 취득하였습니다
