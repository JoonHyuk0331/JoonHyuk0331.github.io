<!--more-->

셀레니움을 활용해 네이버 트렌드 차트를 크롤링하는 프로젝트를 진행한 경험이 있습니다. 이 프로젝트의 목표는 실시간 검색어와 인기 키워드를 자동으로 수집하여 분석하는 것이었습니다. 먼저, 셀레니움을 이용해 네이버 트렌드 페이지에 자동으로 접속하고, 원하는 데이터가 포함된 HTML 요소를 추출했습니다. 이를 위해 다양한 웹 요소를 탐색하고, XPath와 CSS 셀렉터를 사용해 데이터를 정확하게 크롤링하는 방법을 익혔습니다. 이후 크롤링한 데이터를 CSV 파일로 저장하거나 데이터베이스에 저장하는 과정을 구현해, 나중에 데이터를 분석할 수 있도록 했습니다. 또한, 크롤링 시 발생할 수 있는 차단 문제를 해결하기 위해 적절한 대기 시간 조절 및 헤더 설정 등을 사용하여 봇 탐지 방지를 적용했습니다. 이 프로젝트를 통해 웹 크롤링의 기초와 자동화 기술을 실무에 응용할 수 있는 경험을 쌓았습니다.
